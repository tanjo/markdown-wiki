# markdown-wiki

Markdown を利用した Wiki です.

## 実行

```
npm install
npm start
```

## 主な利用ライブラリ

- [express](https://www.npmjs.com/package/express)
- [marked](https://www.npmjs.com/package/marked)
- [mongoose](https://www.npmjs.com/package/mongoose)
